  • Lesson Learned

    beberapa materi yang sudah dipelajari selama perkuliahan SIPI

  • BPMN (Business Process Model and Notation)

    bahasa pemodelan grafis yang digunakan untuk menggambarkan dan memvisualisasikan proses bisnis. BPMN menggunakan simbol-simbol standar yang mudah dipahami untuk mewakili berbagai elemen dalam proses bisnis, seperti aktivitas, keputusan, aliran, dan gerbang.

  • DFD (Data Flow Diagram)

    Diagram alir data adalah suatu diagram yang menggambarkan aliran data dari sebuah proses atau sistem. DFD juga menyediakan informasi mengenai luaran dan masukan dari setiap entitas dan proses itu sendiri.

  • ERD (Entity Relationship Diagram)

    diagram yang digunakan untuk perancangan suatu database dan menunjukan relasi antar objek atau entitas beserta atribut-atributnya secara detail.

  • HTML

    merupakan salah satu bahasa pengkodean atau pemograman yang digunakan untuk membuat halaman website yang ditampilkan pada web browser. Sebagian besar halaman yang kamu temukan pada internet kebanyakan menggunakan Bahasa HTML.

  • CSS

    CSS digunakan untuk menentukan gaya dari tampilan website yang akan kamu buat seperti tata letak halaman, warna , dan font. Semuanya dapat ditentukan oleh CSS, CSS bertujuan agar website terlihat lebih menarik.
